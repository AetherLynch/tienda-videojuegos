# Tienda de Videojuegos

Actividad remedial de la Unidad 2 de la clase de *Desarrollo Web Integral*.

Este proyecto es una pequeña tienda de videojuegos creada con **HTML**, **CSS** y **JavaScript**, donde se muestra un catálogo dinámico de juegos usando listas, funciones de renderizado y manipulación del DOM.

---

## Características del proyecto

- Catálogo de videojuegos dinámico generado con JavaScript.
- Cada juego muestra:
  - Título
  - Precio
  - Descripción
  - Imagen representativa
- Estilos mejorados con una apariencia profesional en tonos azules.
- Estructura simple y fácil de entender para fines educativos.

---

## Catálogo de Videojuegos

El catálogo se genera dinámicamente desde el archivo `app.js`, donde cada videojuego contiene:

- `title` — Nombre del juego  
- `price` — Precio en MXN  
- `description` — Breve descripción del juego  
- `image` — Ruta de la imagen en la carpeta `/images`  

Las tarjetas del catálogo se muestran en forma de **grid responsivo**, con un diseño azul moderno y sombras suaves.
